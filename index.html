Deploy to GitHub Pages

Publish Your Dashboard

Host this dashboard for free on GitHub Pages in under 5 minutes.

STEP 01

Create Repository

Make a new GitHub repo named lung-cancer-dashboard

STEP 02

Upload Files

Add index.html and survey_lung_cancer.csv to the repo root

STEP 03

Enable Pages

Go to Settings → Pages → Source: main branch → Save

STEP 04

Go Live

Your site will be live at username.github.io/lung-cancer-dashboard

# Terminal commands
git init
git add index.html survey_lung_cancer.csv
git commit -m "feat: add lung cancer dashboard"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/lung-cancer-dashboard.git
git push -u origin main
